Границя

Границя — це фундаментальне поняття математичного аналізу, яке використовується для визначення поведінки функції або послідовності при наближенні аргументу до певного значення.

Означення

Означення: Якщо для будь-якого малого додатного числа ε існує таке δ, що для всіх x, які задовольняють |x - a| < δ, виконується |f(x) - L| < ε, то L називається границею функції f(x) при x → a.
